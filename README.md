# steven

Drinking game for nerds


## Flow
1. Det bliver sin tur, alle kan se et "skjult kort".
2. Man vender kortet, alle kan se kortet
3. Efter man har gjort hvad der skal gøres skal man trykke "pass" og det bliver næstes tur.
